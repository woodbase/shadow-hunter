# Xeno Breach – Style Guide

Version: 0.1

This document defines the visual identity of the Xeno Breach project.  
All contributors should follow these guidelines when creating UI, environments, assets, or promotional material.

---

# 1. Design Philosophy

Xeno Breach is **industrial sci-fi horror**.

The world should feel:

- Cold
- Functional
- Used
- Dangerous
- Claustrophobic

Technology in this universe is not elegant or futuristic.

Instead it is:

- utilitarian
- military
- worn down
- built for survival

Avoid:

- cartoon aesthetics
- bright neon cyberpunk visuals
- overly clean sci-fi environments

Preferred inspirations:

- Alien
- Dead Space
- Event Horizon
- SCP Foundation
- Industrial science fiction environments

# 4. UI Design Language

UI elements should feel like part of tactical equipment.

Preferred shapes:

- rectangular panels
- sharp edges
- thin borders

Avoid:

- rounded bubbles
- playful UI elements
- overly decorative graphics

---

# 5. Lighting Style

Lighting should support tension and atmosphere.

General principles:

- environments should remain dark
- strong contrast between light and shadow
- flickering lights are acceptable
- emergency lighting should create tension

Lighting Colors

Normal lighting  
Cold white / blue

Emergency lighting  
Red

Technology glow  
Cyan

Unknown phenomena  
Purple or distorted light

---

# 6. Environmental Tone

Locations should feel like real infrastructure.

Examples:

- research facilities
- industrial corridors
- maintenance tunnels
- reactor rooms
- storage sectors

Materials should show wear:

- scratched metal
- dirt
- oil stains
- industrial markings
- exposed cables

Avoid clean futuristic surfaces.

---

# 7. Animation Style

Animations should feel mechanical and deliberate.

Preferred styles:

- short transitions
- precise motion
- equipment-like movement

Avoid:

- exaggerated animation
- playful motion
- arcade style feedback

---

# 8. Icon Style

Icons should be minimal and functional.

Rules:

- line-based icons
- consistent stroke width
- minimal detail
- clear silhouettes

Examples:

Health icon  
Outline heart

Ammo icon  
Bullet cartridge

Objective icon  
Directional arrow

---

# 9. Overall Design Rule

Every visual decision should reinforce these words:

Industrial  
Cold  
Functional  
Dangerous  
Claustrophobic

If something looks too clean, colorful, or playful, it likely does not belong in Xeno Breach.
